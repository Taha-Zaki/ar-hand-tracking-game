## Overview

ar-hand-tracking-game is an open-source developer tool that integrates real-time hand gesture recognition with immersive augmented reality gameplay. Built with React, Three.js, and MediaPipe, it enables developers to craft interactive AR experiences driven by intuitive hand interactions.

## Installation

Build ar-hand-tracking-game from the source and install dependencies:

1. **Clone the repository:**
   ```bash
   > git clone https://github.com/Taha-Zaki/ar-hand-tracking-game
   ```

2. **Navigate to the project directory:**
   ```bash
   > cd ar-hand-tracking-game
   ```

3. **Install the dependencies:**

Using [npm](https://www.npmjs.com/):

```bash
> npm install
```
